# Sistema de Gestión Vehicular (Java Web)

Aplicación Full-Stack para la gestión de inventario automotriz. Desarrollada con arquitectura MVC, diseño Material You y buenas prácticas de ingeniería de software.

##  Tecnologías (Tech Stack)
* **Backend:** Java 17, JSP, Servlets, JDBC.
* **Base de Datos:** MySQL (con pool de conexiones Singleton).
* **Frontend:** CSS3 (Material Design 3 / Material You), Flexbox, Responsive Design.
* **Herramientas:** Maven, Tomcat 10, Git.

##  Características Clave
1.  **CRUD Completo:** Crear, Leer, Actualizar y Eliminar vehículos.
2.  **Borrado Lógico (Soft Delete):** Seguridad de datos mediante flag en BD (no se borra físicamente).
3.  **Interfaz Moderna:** Estilo "Earthy" con bordes redondeados y paleta de colores profesional.
4.  **UX Mejorada:** Modales de confirmación, badges de estado y generación automática de imágenes placeholder.
